Galleria

Demo: http://www.aswink.in/demo/galleria/

Galleria is a simple image gallery. To use it download a copy of index.php file to the root folder of your server, and your are ready togo!

Note: In order to make the app single-paged, some files are being fetched from remote servers. So it requires an internet connection to function properly. The app makes use of Bootstrap and Google Fonts
